# Data Dictionary

The pipeline expects the following raw inputs in `data/raw/`. All data is
de-identified TCGA HNSC cohort data.

## Raw inputs

| File | Modality | Shape | Key |
|---|---|---|---|
| `clinical.txt` | Clinical | 309 rows → 82 patients | patient barcode |
| `exposure.txt` | Risk factors (smoking/alcohol) | 114 rows → 82 patients | patient barcode |
| `follow_up.txt` | Longitudinal follow-up | 279 rows → 82 patients | patient barcode |
| `pathology_detail.txt` | Pathology detail | 175 rows → 82 patients | patient barcode |
| `mutations.txt` | Genome (MAF-style) | 13,180 calls, 82 tumor samples | sample barcode |
| `transcriptomics.txt` | Transcriptome (RNA-seq) | 20,502 genes × 92 samples | gene × sample |
| `methylation.txt` | Epigenome (methylation) | 20,114 probes × 96 samples | probe × sample |

## Processed outputs

Generated by `src/build_dataset.py` into `data/processed/`:

| File | Purpose |
|---|---|
| `clinical_master.csv` | Merged patient-level clinical table |
| `genome_features.csv` | Per-sample mutation-derived features |
| `transcriptome_matrix.csv` | Sample × gene expression matrix |
| `methylation_matrix.csv` | Sample × probe methylation matrix |
| `task1_tumor_vs_normal.csv` | Modeling table for Task 1 |
| `task2_stage.csv` | Modeling table for Task 2 |
| `task3_risk_factors.csv` | Modeling table for Task 3 |
| `task4_survival.csv` | Modeling table for Task 4 |

## Barcode convention

TCGA sample barcodes encode sample type in the suffix:
`-01` = primary tumor, `-11` = matched normal tissue. See
`src/utils/barcode.py`.

> **Note:** raw files are not committed by default. Place your own TCGA HNSC
> exports in `data/raw/` before running the pipeline.
